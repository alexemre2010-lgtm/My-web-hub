# Personal Link Hub

A simple and colorful personal link hub web page inspired by Linktree.

## Features
- Centered profile card
- Profile image
- Two social links for Instagram and GitHub
- Dark/light mode toggle
- Wacky, playful visual style

## Files
- `index.html` — main page structure
- `styles.css` — styling and theme behavior

## How to view it
1. Open `index.html` in your browser.
2. Or use a simple local server if you want live preview support.

## Notes
You can edit the profile name, bio, image, and links directly in `index.html`.
